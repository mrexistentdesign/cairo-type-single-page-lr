## Cairo-Variable-Type
This was my first time learning how to apply variable fonts, footers, and navigation bars, with this being learned in Web Typography Class.

## Font Used
Cairo - Variable

## Process
The main goal I had going into this project was the idea of making most of the designs out of just letter forms. This affected the choice of variable font for this project, as Cairo supports the Arabic lexicon, allowing for more unique shapes to be used. Since Cairo is a place in egypt, and the font was intended to be multi-lingual, it only made sense to depict the warm colors of the dunes in a sunset, contrasting with a deep blue representing rivers. Since most of the design was text based, the SVGs were added last as background images, as it would only be possible to crop them correctly if everything else was in a proper grid.

## Reflection
The main people I would want to use this site would be designers like me. That is why I decided on making most of the elements more abstract, letting the main letterforms shine through. By doing that, it lets the design stand on the Cairo font itself, which is important for a site that is meant to show it off. Everything is all seperated meaningfully as well, with colors being used intentionally to create wide pockets of negative space, making it easier to find each section, as well as it being inside an (optically) one column grid. The only thing I would improve if given more time to work on this project is helping make the grid work better in mobile, as some assets refused to move within the grid without manual placement.
